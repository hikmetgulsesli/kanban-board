# Kanban Board

React + Vite + Tailwind CSS ile geliştirilmiş modern Kanban Board uygulaması.

## Özellikler

- Sürükle-bırak ile kart taşıma
- Kolon yönetimi (ekleme, silme, yeniden sıralama)
- Kart CRUD (başlık, açıklama, etiketler, renk kodları, son tarih)
- Filtreleme ve arama
- LocalStorage ile veri kalıcılığı
- Responsive tasarım

## Teknolojiler

- React 19
- Vite 7
- Tailwind CSS 3
- TypeScript
- Lucide React (icons)

## Geliştirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
